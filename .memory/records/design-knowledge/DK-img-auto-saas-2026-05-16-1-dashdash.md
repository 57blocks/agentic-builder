# Style Spec — auto-saas-2026-05-16-1-dashdash.png

**Industry**: saas

**Summary**: A clean, spacious SaaS landing/workspace entry screen with soft neutrals, understated borders, and a few warm accent moments. The interface feels productivity-focused and approachable, with lightweight cards, pill chips, and restrained typography.

### Palette
- Primary: `#f6b24d` — warm amber
- Secondary: `#4f022f` — deep plum
- Accent: `#ffd48a` — soft peach gradient
- Background: `#fbfbf9` — off-white canvas
- Surface: `#ffffff` — white panels
- Text: `#202125` — charcoal
- Text muted: `#7c7d81` — muted gray
- Border: `#e4e5e7` — light gray border
- Warning: `#f6e8b3` — pale yellow
- Danger: `#f06a6a` — soft coral

### Typography
- Heading font: Inter (weight 600)
- Body font: Inter (weight 400)
- Base size: 16px
- Note: large centered hero heading
- Note: lightweight secondary navigation text
- Note: muted placeholder text
- Note: small uppercase-free pill labels

### Spacing & Radius
- Spacing base: 8px; scale: 4, 8, 12, 16, 24, 32, 48, 64
- Radius: sm 6px, md 10px, lg 16px, pill 999px
- Shadows: 2 variant(s)

  - `0 8px 24px rgba(0,0,0,0.06)`

### Components
- **button**: Mostly text-first buttons with minimal chrome; primary action is a compact rounded-square send button using a peach-to-amber gradient, while secondary auth actions are subtle ghost buttons with light borders.
- **card**: Template suggestions appear as tiny pill cards with white backgrounds, thin gray borders, small leading icons, and generous horizontal padding; cookie notice uses a tinted warning surface with a soft yellow border.
- **input**: Large prompt textarea centered on the page with white fill, thin amber focus border, soft rounded corners, muted placeholder text, and trailing utility/send icons inside the field.
- **navigation**: Top navigation is a slim header with a dark promotional banner above it, left-aligned title/back affordance, and right-aligned auth links; footer navigation is sparse and low-contrast.

### Layout
promo banner + slim top nav + centered hero prompt input + four-column template chip grid + sparse footer + bottom-left cookie notice
